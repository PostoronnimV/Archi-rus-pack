Системный софт (system software) представляет программную среду для отдельных типов программ и данных, которые размещены в ней в форме информобъектов.

Системный софт является специализацией "железа", которое используется для моделирования программной среды, в которой работают информобъекты. Системный софт также может быть использован для того, чтобы показать коммуникационных брокеров (middleware).

Обычно, системный софт комбинируется с устройством, представляющим аппаратуру, и образует с ним общее "железо".

Для системного софта может быть назначено устройство (но не наоборот!!!).

Системный софт может быть назначен на информобъект (который на нём разворачивается. Но не наоборот!)

"Железо" может содержать системный софт.

Имя системного софта должно быть предпочтительно существительным, ссылающимся на тип вычислительной среды, например, "J2EE сервер".

Системный софт может содержать другой системный софт, например операционная система может содержать базу данных.

Категория:

"Железов". Выполнитель.

Примеры:

DBMS, DB2 System Software, обработка очередей сообщений, операционная система, сервер приложений J2EE

Типовые отношения:

Назначается - для устройства, информобъекта (но не наоборот)
